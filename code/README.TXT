PROJECT TITLE:
Minimarket Management System

PURPOSE OF PROJECT:
This Java project aims to simulate the management system of a minimarket. It includes classes for minimarket operations, employees (Pegawai), customers (Pembeli), and various types of products (Produk, Kecantikan, Konsumsi, Pembersih).

VERSION or DATE: 
5 Nov 2023

HOW TO START THIS PROJECT:
1. Ensure you have a Java development environment set up.
2. Compile and run the 'Minimarket' class to start the minimarket simulation.


AUTHORS:

USER INSTRUCTIONS:
- Run the 'Minimarket' class to start the simulation.
- Follow the prompts to add employees, list products, and perform transactions.
- Explore the different product types: Kecantikan, Konsumsi, Pembersih.
- Enjoy managing your virtual minimarket!
